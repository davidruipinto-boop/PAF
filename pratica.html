Enunciados práticos para simular

C - Simulação 1: alunos e notas

Cria um programa que leia número, nome, nota 1 e nota 2 de vários alunos até número 0. Mostra média e estado: Aprovado se média ≥ 10, Reprovado se média < 7, em Prova Final nos restantes casos.

Treina: structs, arrays, ciclo while, funções e média.

C - Simulação 2: pedido de compra com IVA

Lê produtos de um pedido até código 0. Cada produto tem código, nome, preço unitário e quantidade. Mostra subtotal e total com IVA de 23%.

Treina: estruturas, acumuladores, funções e formatação.

C - Simulação 3: ficheiro CSV

Regista alunos num ficheiro CSV com matrícula, nome, código de curso e saldo. Depois lê o ficheiro e mostra maior saldo, menor saldo e média.

Treina: FILE, fopen, fprintf, fscanf, fclose e structs.

Java - Simulação 1: sistema de iluminação

Cria a classe Iluminacao com turnOn(), turnOff() e mostraEstado(). A aplicação AppIluminacao deve ter janela com 6 divisões e botões para ligar/desligar.

Treina: classe, objetos, arrays, Swing e ActionListener.

Java - Simulação 2: gestão de conta bancária

Cria Conta com inserir(), depositar(), levantar(), verificarSaldo() e mostrar(). A janela deve criar conta e permitir movimentos.

Treina: POO, validação, try/catch, separação classe lógica / classe janela.

HTML - Simulação: site da avaliação “Andar de Mota”

Cria um site com pelo menos três páginas: Início, Galeria e Tabela. Deve ter menu de navegação, galeria de imagens com legendas, uma tabela, um vídeo, um áudio e texto claro. Usa Bootstrap para estilos e não uses JavaScript personalizado.

Consulta: abre a ficha Bootstrap baseada no teu site e confirma que cumpre: 3 páginas, vídeo, áudio, tabela, galeria, navegação e Bootstrap.

SQL/PHP - Simulação: formulário com MySQL

Cria base de dados, tabela, formulário HTML/PHP para inserir dados e página para listar registos. Usa validação simples e prepared statements.
